== Installation

  git clone git://github.com/tetsuo692/dotfiles ~/.dotfiles
  add following lines in .bashrc:
    . ~/.dotfiles/bashrc
  and this in .bash_profile
    if [ -f ~/.bashrc ];
      then
        source ~/.bashrc
    fi
